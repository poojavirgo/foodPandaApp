Splash!

CTEC3905 slideshow viewer

This is a demonstration project for CTEC3905. It contains all the slides for the module. The index presentation can be accessed using the home link at the top of the page. It contains a slide with a link for each presentation in the system.

Swipe or use left/right arrow keys to view next/previous slides. Or use the buttons at the bottom.
